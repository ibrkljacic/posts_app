# Space

Specifications for space units based on `4px`.

<Canvas>
  <Space />
</Canvas>

## Notes

- Avoid hardcoding space in UIs (e.g. `margin`, `padding`).
- For non-conforming space (e.g. `'1px'`, `2px`, `3px` etc), hardcode the pixel values after confirmation with design. These should ideally exist as private implementations of components and not liberally used in business logic and page layouts.
