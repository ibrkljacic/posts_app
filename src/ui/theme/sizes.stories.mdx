# Sizes

Specification for various hardcoded and shared sizes.

## Notes

- Avoid hardcoding sizes in the UI and instead discuss and introduce it in the `Sizes` spec.
- Try to share sizes, whenever applicable, if the semantic size name makes sense for the applied size.
