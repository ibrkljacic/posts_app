# Menu

`Menu` is an input control that is used to present selectable options in a dropdown form.

## Props

`Menu` needs an anchor element and the set of options.

<ArgsTable of={Menu} />

## Use

<Canvas>
  <FlexLayout justifyContent="flex-end" sx={{ height: '200px' }}>
    <Menu
      anchorElement={<Icon icon="menu" size="l" />}
      options={[
        { icon: 'user', text: 'My Profile', onClick: () => alert('My Profile') },
        { icon: 'logOut', text: 'Log Out', onClick: () => alert('Log Out') },
      ]}
    />
  </FlexLayout>
</Canvas>
