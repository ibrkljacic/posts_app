# TextInput

`TextInput` an extension of `HTML` input element that provides a consistent way to render a text input component in UI system.

## Props

`TextInput` is a controlled component which exposes `value` and `onChange` props. <br/>
Also, it is wrapped by `withLabel` HOC that exposes `label` prop.

<ArgsTable of={TextInput} />

## Use

Easily render input component and control it with `value` and `onChange` callback which returns newly updated value.

<Canvas>
  <Box sx={{ width: '320px' }}>
    <TextInput label="Label example" value="Fixed value" onChange={alert} />
  </Box>
</Canvas>

### Variants

`TextInput` supports the following variants to ensure consistent rendering and usage of text inputs in UIs. Each variant applies the state behaviors (mentioned in earlier sections) consistently.
