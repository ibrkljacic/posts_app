# Switch

`Switch` is an input control that is used to quickly switch between two possible states.

## Props

`Switch` is a controlled component which exposes `value` and `onChange` props.

<ArgsTable of={Switch} />

## Use

Easily render switch component and control it with `value` and `onChange` callback which returns newly updated value.

<Canvas>
  <Switch label="Active" value={true} onChange={() => {}} />
</Canvas>

## Behaviors

`Switch` supports a number of useful behaviors usable in various UI layouts and features.
