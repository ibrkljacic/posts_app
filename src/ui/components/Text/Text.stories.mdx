# Text

The `Text` component enforces the typography styles defined in the **Typography** spec.

All text content in the **UI System** should use the `Text` component directly or indirectly.

## Props

`Text` supports all core props that the `Box` element supports, and other relevant typography-based properties.

<ArgsTable of={Text} />

## Use

### Basic

Apply supported colors and typography styles using the `color` and `variant` prop. You can also apply theme props accordingly.

<Canvas>
  <Text bg="gray-700" color="white" px={4} py={3} sx={{ borderRadius: 's' }} variant="m-spaced-bold">
    This is a m-spaced-bold text with white color, with applied theme props.
  </Text>
</Canvas>

### Color

`Text` is theme-aware, and accepts theme-based color values

<Canvas>
  <FlexLayout space={4}>
    <Text color="red-500">red-500</Text>
    <Text color="gray-700">gray-700</Text>
    <Text color="black">black</Text>
  </FlexLayout>
</Canvas>

### Variants

`Text` applies supported typography styles through the `variant` prop.

<Canvas>
  <FlexLayout flexDirection="column" space={4}>
    {Object.keys(theme.text).map((variant) => {
      return (
        <Text color="black" key={variant} variant={variant}>
          {capitalize(variant)}
        </Text>
      );
    })}
  </FlexLayout>
</Canvas>

## Behaviors

`Text` supports a few simple ways to customize behaviors of text content.

### `as` another component

`Text`, like all **Core** components, supports the ability to render as another element using the `as` prop. By default, `Text` is rendered as an inline `span` element, and it could be useful to set this to a `div` element for applying block-based layout display. This is useful to leverage `Text`'s theme props and apply them to semantic HTML elements.

Check the browser inspector to observe final rendering results in the following examples.

<Canvas>
  <FlexLayout flexDirection="column" space={4}>
    <Text as="div" color="red-500" variant="m-spaced-bold">
      As a div element
    </Text>
    <Text as="strong" color="red-500" variant="m-spaced-bold">
      As a strong element
    </Text>
    <Text as="blockquote" color="red-500" variant="m-spaced-bold">
      As a blockquote element
    </Text>
    <Text as="li" color="red-500" variant="m-spaced-bold">
      As a list item element
    </Text>
  </FlexLayout>
</Canvas>
