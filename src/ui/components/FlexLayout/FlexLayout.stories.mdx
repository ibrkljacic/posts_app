# FlexLayout

`FlexLayout` provides a simple but powerful way to build flex-based layouts with consistent spacing between child elements. Spacing units are defined in the **Space** spec.

## Props

`FlexLayout` supports all core props that the `Box` element supports, and other relevant flex-based properties.

<ArgsTable of={FlexLayout} />

## Use

With `FlexLayout`, you can build any advanced layout with ease and consistency! The code is also readable and represents the intention of the desired layout.

<Canvas>
  <FlexLayout bg="gray-400" justifyContent="space-between" px={5} py={3} space={5}>
    <FlexLayout alignItems="center" space={5}>
      <Icon color="gray-600" icon="search" />
      <Text variant="m-spaced-medium">Search</Text>
    </FlexLayout>
    <FlexLayout space={5}>
      <FlexLayout alignItems="center" flexDirection="column" space={1}>
        <Icon icon="user" />
        <Text variant="s-spaced-bold-caps">Account</Text>
      </FlexLayout>
      <FlexLayout alignItems="center" color="red-500" flexDirection="column" space={1}>
        <Icon icon="chevronDown" />
        <Text variant="s-spaced-bold-caps">Notifications</Text>
      </FlexLayout>
    </FlexLayout>
  </FlexLayout>
</Canvas>
